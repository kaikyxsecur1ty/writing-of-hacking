# introdução à comandos básicos do terminal do linux

## Comandos básicos essenciais
----
- sempre que tiver dúvidas sobre as opções do comando ou quer saber do que determinado comando é capaz use:
  - use: `man [COMANDO]`
  - exemplo: `man ls` : *o comando `ls` será apresentado em breve.*
  - este comando `man` significa **manual.**
